# 构建内在秩序 自爱显化 - 个人网站

> 假我旧版本清洗与真我新版本认同课程官网

## 🚀 项目简介

这是"构建内在秩序 自爱显化"课程的官方网站，帮助学员从被限制性信念困住、情绪冻结僵住、找不到自己主体性的状态，转变为情绪自由流动、建立能量边界、活出高能量状态的全新版本。

## ✨ 核心功能

- 📚 **四阶段蜕变路径** - 系统性的成长路径
- 💔 **转变对比** - 看见改变的力量
- 🧘‍♀️ **12个专项冥想** - 专业的冥想工具包
- 🎨 **精美设计** - 温暖、疗愈、有力量感的视觉体验
- 📱 **响应式设计** - 完美适配手机和电脑

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- 无需框架，轻量快速
- Cloudflare Pages 自动部署
