MODEL ACCESS

选择溯光 Agent 的模型额度

溯光负责 Agent 的角色、方法论、训练资料和对话流程。大模型算力可以使用官方体验额度，也可以接入用户自己的 API Key。

使用方式

当前使用官方体验模式。自带 Key 模式下，Key 只保存到本机浏览器。

收费 MVP 的额度边界

每个产品档位都有每日消息数、估算 token 和单次输入长度限制。当前版本会在浏览器本地记录用量；部署到 Cloudflare 时也可以绑定 `SUGUANG_USAGE_KV` 做服务端日限额。

  • 官方额度：平台承担 token 成本，按产品档位封顶。
  • 自带 Key：用户承担模型成本，仍保留输入长度和安全边界。
  • 超限后：前端停止远程调用，回退本地规则并引导升级或人工跟进。

SAFETY & PRIVACY

使用边界与隐私说明

溯光 Agent 用于成长陪伴、服务匹配和初步梳理。它不能替代正式心理咨询、医疗诊断、危机干预或法律/投资建议。

AI 使用边界

Agent 可以做情绪承接、五步法梳理、潜意识练习简版引导、服务路径匹配和预约引导，但不会承诺疗效，也不会替你做重大人生决定。

危机情况

如果你有自伤、自杀、活不下去或伤害他人的想法，请优先联系当地紧急救助、可信任的人或专业危机热线。Agent 不能处理即时危机干预。

API Key

自带 Key 模式下，Key 保存在当前浏览器。正式部署时，请通过 `/api/chat` 临时转发给模型服务商；溯光不应明文长期保存用户 Key。

预约资料

当前 MVP 的预约意向保存在本地浏览器，并生成跟进摘要，可导出给人工处理。正式上线建议接入账号、CRM、隐私协议、删除机制和服务端加密。
