
# 溯光AI知识库 - 潜意识练习

## 练习1：勾勒理想场景潜意识练习
**文件路径：** assets/exercises/勾勒理想场景潜意识练习.pages
**用途：** 通过五感锚定，构建并植入理想未来场景，改写潜意识剧本
**适用：** 目标显化、愿景规划、自我突破
**核心要素：**
- 进入阿尔法/theta状态
- 五感构建：视觉、听觉、触觉、嗅觉、味觉
- 情绪放大：喜悦、感恩、平静
- 锚定动作：呼吸、手势
- 日常强化：睡前/醒来回顾

## 溯光AI使用指南
