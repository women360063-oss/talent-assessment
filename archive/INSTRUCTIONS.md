# 天赋测评网页 - 部署指南

## 🚀 最简单的方法：Cloudflare Pages 直接上传

### 第一步：获取 index.html 文件

文件位置：`/root/.openclaw/workspace/talent-assessment/index.html`

### 第二步：上传到 Cloudflare Pages

1. 访问 https://pages.cloudflare.com
2. 登录：women360063@gmail.com
3. 点击 "Create a project"
4. 选择 "Upload assets"
5. 项目名称：talent-assessment
6. 上传 index.html 文件
7. 点击 "Deploy site"

### 第三步：绑定域名 healwealthy.com

1. 部署成功后，进入项目设置
2. 点击 "Custom domains"
3. 添加域名：healwealthy.com
